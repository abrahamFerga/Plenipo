## Usage

When the user asks for user-facing text in the organization's voice:

1. Read `references/voice-guide.md` for the tone rules and the terminology table.
2. Draft (or rewrite) the text applying every rule.
3. If any banned term appears in the source text, replace it per the table and mention the
   replacements you made.

Keep the result the same length or shorter than the source unless the user asks for expansion.
