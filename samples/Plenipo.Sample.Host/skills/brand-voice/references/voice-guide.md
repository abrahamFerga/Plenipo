# Voice guide

## Tone rules

1. Direct and warm; no corporate filler ("leverage", "synergy", "circle back").
2. Short sentences. One idea per sentence.
3. Active voice; the reader is "you", we are "we".
4. Numbers as digits, dates as `15 March 2026`.

## Terminology

| Banned | Use instead |
|---|---|
| utilize | use |
| leverage | use |
| going forward | from now on |
| ASAP | by <specific date/time> |
| end user | customer |
